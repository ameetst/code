# Mutual Fund Analysis Pipeline

A comprehensive Python-based pipeline for analyzing Indian mutual funds' historical performance using AMFI (Association of Mutual Funds in India) data.

## Overview

This project consists of three interconnected scripts that extract ISIN codes for mutual funds, fetch historical NAV data, and calculate comprehensive performance metrics with comparative visualizations.

---

## Pipeline Architecture

```
get_isin_codes.py
    ↓ (outputs: {category}_isin_codes.csv)
    ↓
get_nav_history.py
    ↓ (outputs: historical_nav_10_years.csv)
    ↓
calculate_metrics.py
    ↓ (outputs: fund_metrics_summary.csv, top5_funds.csv, fund_plots/)
```

---

## Script 1: `get_isin_codes.py`

### Purpose
Extracts ISIN codes for mutual funds from AMFI's official NAV data, filtered by fund category (e.g., midcap, flexicap, smallcap) and plan type (direct plans only).

### Key Features
- Filters funds by **category** (comma-separated list supported for multiple categories)
- Filters for **direct plans only** (lower expense ratios)
- Handles multiple ISIN types: Growth (Div Payout) and Reinvestment
- Removes invalid/empty ISIN codes
- Outputs clean CSV with fund names and ISINs

### Configuration
```python
categories_input = 'smallcap'  # Change to desired categories
# Examples: 'midcap', 'flexicap', 'largecap', 'midcap, flexicap'
```

### Usage
```bash
python get_isin_codes.py
```

### Inputs
- AMFI NAVAll.txt (fetched automatically from: https://www.amfiindia.com/spages/NAVAll.txt)

### Outputs
- **{category}_isin_codes.csv**: Semicolon-separated file with columns:
  - MF NAME: Fund scheme name
  - ISIN CODE: Unique ISIN identifier

### Example Output
```
Aditya Birla Sun Life Midcap Fund - Growth - Direct Plan;INF209K01LX6
HDFC Midcap Fund - Direct Plan - Growth;INF174KA1LV9
Axis Midcap Fund - Direct Plan - Growth;INF846K01AS7
```

## Script 2: `get_nav_history.py`

### Purpose
Downloads 10 years of historical daily NAV (Net Asset Value) data for selected mutual funds using the mfapi.in API and AMFI Scheme Code mapping.

### Key Features
- Maps ISIN codes to AMFI Scheme Codes
- Fetches complete historical NAV from mfapi.in
- Filters data for past 10 years
- Combines data from multiple ISINs into a single dataset
- Robust error handling for network/API failures

### Configuration
```python
input_csv = 'smallcap_isin_codes.csv'  # Match output from get_isin_codes.py
output_csv = 'historical_nav_10_years.csv'
```

### Usage
```bash
python get_nav_history.py
```

### Inputs
- **{category}_isin_codes.csv** (generated by get_isin_codes.py)
- AMFI NAVAll.txt (fetched automatically for Scheme Code mapping)
- mfapi.in API (for historical NAV data)

### Outputs
- **historical_nav_10_years.csv**: CSV with columns:
  - Date: NAV date (YYYY-MM-DD format)
  - MF NAME: Fund scheme name
  - ISIN CODE: ISIN identifier
  - nav: Daily NAV value

### Example Output
```
Date,MF NAME,ISIN CODE,nav
2016-04-14,Aditya Birla Sun Life Midcap Fund - Growth - Direct Plan,INF209K01LX6,10.0482
2016-04-15,Aditya Birla Sun Life Midcap Fund - Growth - Direct Plan,INF209K01LX6,10.0501
```

## Script 3: `calculate_metrics.py`

### Purpose
Calculates comprehensive performance metrics for all funds and generates comparative visualizations for the top 5 performing funds.

### Key Features
- **Metrics Calculated**:
  - Overall CAGR (Compound Annual Growth Rate)
  - 1-Year, 3-Year, 5-Year CAGRs
  - Volatility (Annualized)
  - Sharpe Ratio (Risk-Free Rate: 6.5%)
  - Maximum Drawdown
  - Rolling 1-Year, 3-Year, 5-Year CAGRs

- **Visualizations** (for top 5 funds):
  - Normalized NAV comparison chart
  - CAGR bar chart
  - Volatility bar chart
  - Max Drawdown bar chart
  - Rolling 1-Year CAGR line chart
  - Rolling 3-Year CAGR line chart
  - Rolling 5-Year CAGR line chart

### Configuration
```python
input_csv = 'historical_nav_10_years.csv'  # Input NAV data
output_csv = 'fund_metrics_summary.csv'    # All funds metrics
RFR = 0.065  # Risk-Free Rate (6.5%) for Sharpe Ratio calculation
```

### Usage
```bash
python calculate_metrics.py
```

### Inputs
- **historical_nav_10_years.csv** (generated by get_nav_history.py)

### Outputs
- **fund_metrics_summary.csv**: All funds ranked by Overall CAGR with columns:
  - MF NAME
  - Start Date / End Date
  - Total Years (of data)
  - Initial NAV / Final NAV
  - Overall CAGR (%)
  - Volatility (%)
  - Sharpe Ratio
  - Max Drawdown (%)
  - 1Y CAGR (%) / 3Y CAGR (%) / 5Y CAGR (%)

- **top5_funds.csv**: Top 5 performing funds (by Overall CAGR) with full metrics

- **fund_plots/** Directory containing:
  - `top5_nav_comparison.png` - Normalized NAV comparison
  - `top5_cagr_comparison.png` - Overall CAGR bar chart
  - `top5_volatility_comparison.png` - Volatility comparison
  - `top5_drawdown_comparison.png` - Max Drawdown comparison
  - `top5_rolling_1yr_cagr.png` - Rolling 1-year CAGR trends
  - `top5_rolling_3yr_cagr.png` - Rolling 3-year CAGR trends
  - `top5_rolling_5yr_cagr.png` - Rolling 5-year CAGR trends

### Example Metrics Output
```
MF NAME,Start Date,End Date,Overall CAGR (%),Volatility (%),Sharpe Ratio,...
Union Midcap Fund - Direct Plan - Growth Option,2020-03-30,2026-04-13,30.29,16.38,1.45,...

```

## Key Metrics Explained

### CAGR (Compound Annual Growth Rate)
Returns the annual growth rate of an investment over the entire period, accounting for compounding.
- Formula: `((Final NAV / Initial NAV) ^ (1 / years)) - 1`

### Volatility
Annualized standard deviation of daily returns, measuring price fluctuation and risk.
- Formula: `std_dev(daily_returns) × √252`

### Sharpe Ratio
Risk-adjusted return metric showing excess return per unit of risk.
- Formula: `(Annual Return - Risk-Free Rate) / Volatility`
- Higher values indicate better risk-adjusted returns (>1 is good)

### Maximum Drawdown
Largest peak-to-trough decline, showing worst-case scenario loss.
- Negative value: `(Trough NAV - Peak NAV) / Peak NAV`

### Rolling CAGR
CAGR calculated over a moving time window (1-year, 3-year, 5-year), showing performance trends over time.

## Tips & Best Practices

1. **Category Filtering**: Use comma-separated values for multiple categories
   ```python
   categories_input = 'midcap, flexicap, largecap'
   ```

2. **Direct vs Regular Plans**: The scripts filter for direct plans only (lower costs)

3. **Data Volume**: Large datasets (100+ funds) may take time to process. Progress is printed to console.

4. **API Rate Limits**: mfapi.in may have rate limits; if errors occur, re-run the script.

5. **Date Format**: All outputs use YYYY-MM-DD format for consistency

6. **Risk-Free Rate**: Adjust the RFR value in calculate_metrics.py if needed
   ```python
   RFR = 0.065  # 6.5% (current typical rate)
   ```

## Troubleshooting

### get_isin_codes.py
- **No output file**: Check internet connection; AMFI site may be down
- **Empty results**: Verify category name is spelled correctly

### get_nav_history.py
- **Missing scheme codes**: Some newer/old funds may not have API data
- **Network errors**: mfapi.in may be temporarily unavailable; retry later
- **Input file not found**: Ensure get_isin_codes.py ran successfully

### calculate_metrics.py
- **Fewer than 5 funds**: Some funds may have <1 year of data and are skipped
- **NaN values**: Funds with insufficient data show NaN for certain metrics
- **Graph generation slow**: Reduce the number of funds or close other applications
